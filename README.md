#  IMDB Sentiment Analysis

This project is a Natural Language Processing (NLP) model that classifies IMDB movie reviews as **positive** or **negative**.

---

## Project Overview

The goal of this project is to build a machine learning model that can understand movie reviews and predict their sentiment.

- Input: Movie review text
- Output: Positive / Negative sentiment

---

## Technologies Used

- Python 3.11
- Pandas
- NumPy
- Scikit-learn
- PyTorch (or TensorFlow if used)
- NLTK / Tokenization tools

---

##  Dataset

The dataset contains IMDB movie reviews labeled as:

- 1 → Positive review
- 0 → Negative review
